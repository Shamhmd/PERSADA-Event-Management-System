# PERSADA Event Management System

## Project Overview
PERSADA Event Management System is a web-based application developed to manage student events, registrations, attendance tracking, and certificate issuance.

## Main Modules
1. Student Management
2. Event Management
3. Event Registration
4. Attendance Tracking
5. Certificate Management

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript
